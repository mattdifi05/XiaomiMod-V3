Backup Kernel

1. Flash TWRP;

2. Backup (Backup - select "Boot", "Dtbo" and swipe);

3. Flash Kernel (Install - Select the Kernel zip and swipe);

4. Reboot to system (Reboot - System);

5. Finish!
