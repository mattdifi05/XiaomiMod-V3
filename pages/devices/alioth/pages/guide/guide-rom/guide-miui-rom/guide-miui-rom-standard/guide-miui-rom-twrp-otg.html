Flash MIUI ROM - TWRP OTG

1. Boot TWRP;

2. Format data (Wipe - Format data);

3. Download ROM and transfer it to your USB;

4. Flash (Install - Select storage - USB-OTG - Select zip and swipe);

5. Reboot to system (Reboot - System);

6. Finish!
