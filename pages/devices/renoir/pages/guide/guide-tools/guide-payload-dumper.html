Payload Dumper

1. Download ROM and Payload Dumper zip;

2. Extract Payload Dumper zip;

3. Extract the file "payload.bin" from the rom and insert it in the folder "input" of the folder Payload Dumper;

4. Run the .exe and wait;

5. In the "output" folder you will find the extracted files;

6. Finish!
